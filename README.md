# lang-tools# lang-tools

A collection of lightweight language processing and translation utilities.

## Features

- **Paraphraser**: Tools for rewriting text in different styles.
- **Translation API**: Simple interfaces for language translation.
- **Web Interface**: A clean HTML/JS frontend to interact with the language tools.

## Usage

Open `index.html` in your browser to access the web-based language tools.

## Structure

- `index.html`: Main web interface.
- `translate-api/`: Backend/API logic for translation services.
- `paraphrase_api_readme.md`: Documentation specific to the paraphrasing component.
